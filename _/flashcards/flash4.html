Page 4 - Cleanup

Now its time make all the "extra" items disappear before we save our template for use. Your file should should look like this before we do this. If not, go back and review the previous three steps.

The first element to cleanup is the text that says Title, composer, and copyright. (you don't have to delete these if you don't want to. Which tool from the Tool menu will we use to erase the text??? If you guessed that you will use the Text tool, you are right. (See, this isn't as complicated as you thought)

Next, click on each items handle and press the backspace or delete button on the keyboard. If you want to save time, select all of these items at the same time by choosing Edit->Select All, then pressing backspace again.

Since we only want to see the staff, clef and notes, we need to find a way to hide the time signature and bare lines. Which tool should we use??? This tool isn't as obvious... I'll give you a hint... Hiding elements is specific to the staff that you are working on... Give up... Think you have it... It is the Staff Tool.

When you select the staff tool, you will see a handel at the upper left corner or the staff. DOuble click there to see the staff attributes dialogue box.

Under Items to display, select only the musical items you want to see (the clef and stems are really the only import ones here) Now click okay and you should see this. If you are ready to go on, click below.
